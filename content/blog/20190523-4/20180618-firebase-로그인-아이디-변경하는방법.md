### command에서 로그인 아이디 변경하는 방법

firebase 를 command에서 사용할 경우 

```
firebase login
```
을 통해 한번 이라도 인증을 하게 되면 인증된 아이디를 자동으로 로그인 하게 된다.

다른 계정으로 firebase에 인증하고싶은경우 

```
firebase login --reauth
```
명령어를 사용하게되면 새로운 윈도우에 구글 로그인 화면이 뜨면서 계정을 선택할 수 있다

끝!
